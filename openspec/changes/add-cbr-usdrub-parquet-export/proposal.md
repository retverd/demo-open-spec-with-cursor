# Change: Сбор курса USDRUB через API ЦБ РФ и сохранение в Parquet

## Why

Нужен первый изолированный, понятный пример “spec → proposal → implementation → archive” на реальной интеграции с внешним источником (ЦБ РФ) и реальным форматом хранения (Parquet).

## What Changes

- Добавляем capability для получения официального курса USD/RUB от ЦБ РФ за последние 7 дней (включая текущий день как последний).
- Добавляем сохранение результата в локальную папку в формате **Parquet**.
- Добавляем unit-тесты и интеграционные тесты, которые ходят в реальные системы (ЦБ РФ).

## Impact

- Affected specs (new): `cbr-usdrub`
- Affected code: отсутствует (на этапе proposal код не пишем)
- Data/Storage: появляется новый выходной артефакт — Parquet-файл с историей курса

## Open Questions

Решено в рамках текущего change:

- Если за `today` нет опубликованного значения, считаем результат **частично успешным** и сохраняем доступные значения.
- Сохраняем Parquet **в текущую папку** (cwd).
- В имя файла добавляем **время запуска** (локальное).
